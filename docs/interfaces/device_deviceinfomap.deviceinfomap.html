Interface DeviceInfoMap

Since EnOcean devices need to be taught-in, they must be stored along with some basic meta data like (at least)

  • their equipment profile (EEP) and
  • the sender id which was used for the teach in. This interface defines how to implement this minimal information as a result of a successful teach-in process. As the name suggests, it is intended to represent a map from an DeviceId to an DeviceInfo.
